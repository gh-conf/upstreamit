CLI tool to configure upstream from parent repository. This works only for forked repositories.

## Install

```
npm install @gh-conf/upstreamit -g
```

## Usage

```js
$ upstreamit

// ✔ Upstream Configured!!!
// ✖ Not a forked repository!!!
```
